# Topic Taxonomy

Use these top-level topics as the default coarse clustering for historically accumulated feeds.

You may add a new top-level topic only when a feed clearly does not fit any existing topic without semantic distortion.

## Default topics

### `ai-and-research`

Use for:

- machine learning papers
- AI systems
- training methods
- research commentary
- technically substantive social posts about AI

### `markets-and-trading`

Use for:

- market microstructure
- trading systems
- execution logic
- signal evaluation
- trading psychology and risk discipline

### `people-and-profiles`

Use for:

- author profile pages
- researcher profiles
- creator identity pages
- account-level background and credibility context

### `community-qa-and-discussion`

Use for:

- Reddit threads
- Hacker News item discussions
- Zhihu questions when the answer ecosystem matters more than a single answer
- public social discussion threads where the community response is central

### `general-reference`

Use for:

- useful articles that do not strongly belong to another topic
- mixed reference material
- sparse pages that are still worth indexing

## When to create a new top-level topic

Create a new topic only if all of these are true:

1. the content would materially degrade an existing topic’s coherence
2. the incoming feeds represent a recurring domain, not a one-off edge case
3. a stable book-like structure under the new topic is plausible
