支持来源·Knowledge Sources

对社区讨论、社交平台、长文发布与通用网页，Tapestry 都提供了来源感知的采集路径，并把异构内容统一整理进同一套 Feed、笔记与门户结构中。

Tapestry supports source-aware ingestion for communities, social streams, long-form publishing, and generic web pages, then normalizes them into one feed, note, and portal model.
